## Social Media APP

Application web permettant aux utilisateurs de s’inscrire, se connecter et interagir via des posts  
(publication, commentaires et likes).

---

## Fonctionnalités

- Inscription et connexion utilisateur
- Authentification par token
- Accès aux pages protégées
- Fil de posts récents
- Création et suppression de posts
- Commentaires sur les posts
- Like / Unlike
- Profil utilisateur (privé et public)

---

## Technologies utilisées

### Frontend
- TypeScript
- Tailwind
- Flowbite

### Backend
- Express.js

### Données
- Données stockées en dur
- Aucune base de données
